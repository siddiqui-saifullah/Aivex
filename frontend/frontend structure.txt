FOLDER DESCRIPTIONS
===================

components/
  └── Reusable React components organized by feature/domain
      - dashboard/: Project list, filters, status badges, modals
      - project/: Project details, members, messaging, actions
      - shared/: Shared UI components (Message, UserTile, etc.)
      - ui/: Basic UI building blocks (Button, Input, Loader)

config/
  └── Application configuration files (API, Socket connections)

constants/
  └── Application constants and options

context/
  └── React Context API for state management (Theme, Toast, User)

hooks/
  └── Custom React hooks for reusable logic

layouts/
  └── Layout wrappers for different pages (App, Auth)

routes/
  └── Routing configuration and protected routes

screens/
  └── Page-level components (Dashboard, Login, Project, etc.)

services/
  └── API service modules for backend communication

utils/
  └── Utility functions and helpers
